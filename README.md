# Kalber Dairy KPI Dashboard

This project is an Excel-based KPI dashboard created to analyze sales, production, and profitability trends across Kalber Dairy products.

The dashboard tracks product performance for milk, yogurt, cheese, cream, juice, and buttermilk using pivot tables, charts, and operational KPIs such as the Sales-to-Production (SP) ratio.

Main analyses include:
- Monthly sales and profit trends
- Product-level profitability comparison
- Production efficiency monitoring
- Cost and operational performance analysis
- Detection of unusual seasonal patterns

Tools used:
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Visualization
